Inventory Management System

As part of our CS 3250 Software Development Methods and Tools class, we developed an Inventory Management System. This program efficiently handles inventory items stored in a SQL database, providing distinct graphical user interfaces (GUIs) for both customer orders and employee interactions. One of the key features is its automatic updating mechanism, ensuring real-time synchronization with the inventory. As well as a financial report for company orders. Throughout the semester-long project, we applied industry best practices and methodologies, including the implementation of JUnit for thorough testing of our methods and local database interactions.

Face Recognition

In our MTH 3140 - Linear Algebra research project, we explored the synergy between mathematical concepts and computer science, specifically emphasizing the foundational role of linear algebra and matrices in machine learning. Using pre-trained code libraries like TensorFlow and Keras, we trained the program to recognize faces and identify individuals. This hands-on exploration deepened my understanding of the mathematical principles underpinning machine learning while providing practical insights into applying linear algebra in real-world scenarios. The project helped me with the significance of mathematical but also highlighted the versatility of tools like TensorFlow and Keras in bringing mathematical concepts to life through coding and experimentation.
