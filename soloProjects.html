Solo Projects

Kings Interiors Website

One of my solo projects involved creating a fully functional website for a drywall company, Kings Interiors , using Ruby on Rails. The website was designed to be responsive, user-friendly, and scalable. It features multiple pages, including "Services," "Portfolio," and "Contact Us." Each page is dynamically generated with Ruby on Rails and incorporates embedded multimedia content for an engaging user experience.

To enhance user engagement, I integrated Google Reviews into the website using the Google Reviews API, which dynamically fetches and displays customer feedback. This functionality showcases my ability to work with APIs and implement third-party integrations seamlessly.

The application also includes a "Contact Me" form that sends email notifications after submission, demonstrating my skills in implementing backend functionalities. This project highlights my proficiency with Ruby on Rails, JavaScript, API integration, and modern web development techniques.
