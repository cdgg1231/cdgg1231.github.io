Solo Projects

Daily Log Reporting System (DLRS) Dashboard

The DLRS Dashboard is a **full-stack web application** designed to automate and centralize the processing of industrial log data. It replaced manual, error-prone Excel log sorting with a modern, secure web interface.

I engineered the **RESTful API backend using Python/Flask** and integrated it directly with a **SQL Server database** via SQLAlchemy. This design ensures **authenticated access** to large log datasets, enabling **authorized end-users** to securely view, search, and filter daily records by multiple parameters (date, user, station). The system also features **PDF and Excel export functionality** for seamless report generation.

Collaborative Knowledge Base Editor

I developed a dedicated internal web application to serve as a **centralized knowledge repository** for the engineering and programming team. This solution focused on collaborative efficiency and user management.

The application requires users to pass a **mandatory login page** and allows multiple programmers to **securely upload, edit, and maintain documentation** and technical reference guides. This project highlights my experience in designing systems with **role-based user authentication, file handling, and version control** to support efficient inter-team collaboration and knowledge sharing.

Kings Interiors Website

One of my solo projects involved creating a fully functional website for a drywall company, Kings Interiors , using Ruby on Rails. The website was designed to be responsive, user-friendly, and scalable. It features multiple pages, including "Services," "Portfolio," and "Contact Us." Each page is dynamically generated with Ruby on Rails and incorporates embedded multimedia content for an engaging user experience.

To enhance user engagement, I integrated Google Reviews into the website using the Google Reviews API, which dynamically fetches and displays customer feedback. This functionality showcases my ability to work with APIs and implement third-party integrations seamlessly.

The application also includes a "Contact Me" form that sends email notifications after submission, demonstrating my skills in implementing backend functionalities. This project highlights my proficiency with Ruby on Rails, JavaScript, API integration, and modern web development techniques.
